ML_Clustering_Model

Unsupervised penguin species clustering with KMeans, achieving 92% purity on the Palmer Penguins dataset.

Python Unsupervised Learning Clustering Data Analysis Visualization

Overview

An unsupervised clustering project that recovers penguin species (Adelie, Chinstrap, Gentoo) from morphological measurements and isotope ratios without ever seeing the true labels during training. KMeans with k=3 achieves a purity score of 0.92, meaning 92% of data points are correctly grouped with members of their own species.

Approach

  • Feature engineering: one-hot encoding for categorical variables, dropping irrelevant metadata and non-discriminative columns (sex, clutch completion)
  • Normalization: StandardScaler to prevent features with larger scales (body mass in grams) from dominating Euclidean distance
  • Missing values: rows dropped rather than imputed — few missing entries, preserves data integrity
  • Cluster selection: Elbow Method on inertia confirms k=3 as optimal, matching the known species count

Evaluation

  • Purity score: 0.92 — strong alignment between unsupervised clusters and true species labels
  • Silhouette score: confirms good cluster cohesion and separation across all three groups
  • Elbow plot: clear elbow at k=3 with diminishing returns beyond
  • Cluster scatterplot: spatial separation on culmen length vs depth reflects real morphological differences
  • Silhouette plot: most points have high positive values, indicating well-placed cluster assignments

Key Design Decisions

  • KMeans over DBSCAN/GMM — data has compact, roughly spherical groups; KMeans is a natural fit
  • Island as a feature — species distribution varies by island, providing useful categorical signal
  • Drop vs impute — few missing rows, so dropping preserves real structure without introducing artificial values
