Decision Tree Regressor for air quality CO prediction with K-Fold cross-validation, achieving R² 0.86.

Python Machine Learning Regression Cross-Validation Data Analysis

Overview

A regression project that predicts carbon monoxide (CO) concentration from environmental sensor data using the Air Quality UCI dataset. A Decision Tree Regressor is trained with 10-Fold Cross Validation to select the optimal depth and avoid overfitting, achieving R² 0.86 on training and R² 0.76 on test with a Pearson correlation of 0.87.

Approach

  • Preprocessing: invalid values (-200) replaced with column mean, irrelevant columns (Date, Time) dropped
  • Data split: 85% train+validation / 15% hold-out test, with StandardScaler normalization on features and target
  • Cross-validation: 10-Fold CV across tree depths 2–15, selecting optimal depth of 7 (104 leaves)
  • Metrics: MSE, R², and Pearson correlation on both training and test sets

Visualizations

  • MSE & R² vs Tree Depth: training and validation curves to analyze under/overfitting
  • Feature importance: ranked bar chart showing each feature's contribution
  • Decision tree export: full tree visualization with splits via Graphviz
  • Predictions vs real values: scatter plot showing model accuracy on test set
  • Residuals analysis: residual plot to check for bias or variance issues

Results

  • Training: MSE 0.135, R² 0.865
  • Test: MSE 0.239, R² 0.756, Pearson r = 0.869
  • Good generalization without severe overfitting — moderate train-test gap
  • Optimal depth of 7 balances model complexity with predictive power
