DL Image Denoising

DL-Image-Denoising

Deep Learning model that removes Gaussian noise from grayscale images using a Convolutional Autoencoder.

Python Deep Learning CNN Autoencoder Image Processing Data Augmentation

Overview

A Denoising Convolutional Autoencoder trained on the Fashion-MNIST dataset (28x28 grayscale images of clothing items). The model learns to reconstruct clean images from inputs corrupted with aggressive Gaussian noise (factor 0.5), achieving a final test loss of ~0.015 MSE.

Model Architecture

Symmetric Encoder-Decoder structure with regularization for stability and generalization:

Encoder: Input(28x28x1) → Conv2D(32) → BatchNorm → Dropout → MaxPool
        → Conv2D(16) → BatchNorm → Dropout → MaxPool → Latent(7x7x16)

Decoder: Conv2D(16) → BatchNorm → Dropout → UpSampling
        → Conv2D(32) → BatchNorm → Dropout → UpSampling
        → Conv2D(1, sigmoid) → Output(28x28x1)

Training Strategy

  • Optimizer: Adam — adaptive learning rates for fast, reliable convergence
  • Loss: MSE — directly penalizes per-pixel reconstruction error
  • Batch Size: 128 — up to 50 epochs with early stopping (patience 5)
  • Noise Factor: 0.5 (aggressive) — forces the model to learn robust features
  • Data Augmentation: rotation, shifting, zooming via ImageDataGenerator
  • Regularization: Dropout (0.25) + BatchNormalization to prevent identity mapping and stabilize gradients

Key Design Decisions

  • Convolutional Autoencoder over dense layers — preserves spatial relationships and reduces parameters
  • Fashion-MNIST over MNIST — more complex textures/edges provide a meaningful denoising benchmark
  • Aggressive noise (0.5) — makes the task non-trivial, preventing the model from relying on near-clean inputs
  • Early stopping — automatically finds the optimal epoch, prevents overfitting without manual tuning

Results

  • Final test loss: ~0.015 MSE — strong pixel-level reconstruction fidelity
  • Visual output: 3-row comparison grid (noisy input → denoised output → original clean image)
  • The model successfully recovers clothing shapes, edges, and textures from heavily corrupted inputs
